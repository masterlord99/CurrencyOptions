Kopule visoko koreliranega para:

Rumena slikica je Gaussova kopula, torej kopula zgrajena na večrazsežni normalni porazdelitvi, rdeča pa je Studentova kopula z 3 stopinjami prostosti. EBA priporoča 3-4 stopinje prostosti. Studentova kopula nam omogoča simulacijo večdimenzionalnih marginalnih kvantilov z težkimi repi. Nadaljne robne porazdelitve si lahko poljubno izbiramo sami, ne glede na izbrano kopulo.

Če želimo primerjati historične (empirične) kopule in simulacijo s kopulami, moramo iz zgodovinskih podatkov izluščiti Brownova gibanja, oceniti njihovo empirično porazdelitev in jih narisati kot kvantile. Žal ta princip deluje samo v 2D (očitni razlogi).

Visoko Koreliran par

par <- c(17,23)

vrsta1 <- data[,par[1]]
vrsta2 <- data[,par[2]]

### potegnemo ven Br.Gib

vrsta1 <- diff(log(vrsta1))
vrsta2 <- diff(log(vrsta2))

BG1 <- vrsta1
BG2 <- vrsta2
cor(BG1,BG2)
[1] 0.6425272
# [1] 0.6425272

### Ustvarimo historično kopulo

N <- length(vrsta1)


vrsta1 <- ecdf(BG1)(BG1)
vrsta2 <- ecdf(BG2)(BG2)


Gauss <- rcopula.gauss(N,cor(data[,par]))
sim1G <- Gauss[,1]
sim2G <- Gauss[,2]

risi(histVrsta1 = vrsta1,histVrsta2 = vrsta2,sim1 = sim1G,sim2 = sim2G,imena=names(data)[par])

Student <- rcopula.t(N,df=1,cor(data.frame(x=BG1,y=BG2)))
sim1S <- Student[,1]
sim2S <- Student[,2]

risi(histVrsta1 = vrsta1,histVrsta2 =vrsta2,sim1 = sim1S,sim2 = sim2S,imena=names(data)[par],rgb=red)
